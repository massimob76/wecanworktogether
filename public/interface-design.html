IPTV Interface Design

Internet tv interface design for various clients. The video interfaces are build on Narrowstep technology which utilizes a video content management system called telvOS. TelvOS allows customers to encode, upload, schedule and manage their video files in an online web application. The web application delivers the video in a web interface to viewers with various functionality allowing viewers to rate, search and engage with the video through various means.

HBO - Movie Portal Website

Interface design for HBO pay-per-view internet movie portal website.

The site would only be available in Hungary and content would be geo-blocked outside Hungary as a trial. The site allows users to switch between 3 movie channels, search video, use category buttons to quickly select movie genres.

The site also features community tools, like ratings, send a link to a friend etc. Movie controls will only appear once the user mouse over on the video. User can also adjust the streaming quality to suit their bandwitdh. Movie can be watched fullscreen.
